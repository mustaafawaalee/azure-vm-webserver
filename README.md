# Azure VM Web Server Project

## Overview
This project demonstrates beginner cloud engineering skills by deploying and securing a Linux virtual machine in Microsoft Azure. The project includes creating the VM, configuring network access, connecting with SSH, installing Nginx, and hosting a custom web page.

## Objective
Build a Linux web server in Azure and document the full setup process.

## Tools Used
- Microsoft Azure
- Ubuntu Linux
- SSH
- Network Security Groups (NSGs)
- Nginx

## Skills Demonstrated
- Azure virtual machine deployment
- Basic cloud networking
- Linux administration
- SSH remote access
- Web server installation
- Troubleshooting

## Project Steps
1. Create a resource group
2. Create a Linux virtual machine
3. Configure inbound network rules
4. Connect to the VM using SSH
5. Install Nginx
6. Test the default web page
7. Replace the default page with a custom page
8. Document the setup with screenshots
